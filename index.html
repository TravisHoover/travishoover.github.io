01 · About

Shipping software for a decade.

I'm a software engineer at Fortress Technology Solutions in Nashville, where I build property technology — Node.js services, React frontends, financial transaction systems, and the AWS infrastructure and CI/CD pipelines that ship it all.

Before that I led backend work on cloud video platforms at Smartvue and Johnson Controls: a Stripe billing system built from scratch, an API gateway connecting a fleet of microservices, and internal tooling that helped carry the company through a $20 million acquisition.

I like owning things end to end — the API, the pipeline that deploys it, and the dashboard that tells me it's still alive at 2 a.m.
